# ar-qr-tracer

Trace physical QR codes in real life using AR

## Features

1.	Dynamic QR loader: pick any QR from Photos/Files.
2.	Pinch to scale: adjust the plane size easily on your wall.
3.	Kanji marker anchor: QR floats above marker; marker can be permanent.
4.	iOS Safari compatible: works on iPhone 14 + iOS 26.
5.	Instructions & File Input layout fixed: no overlap, clean margins.
